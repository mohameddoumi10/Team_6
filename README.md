# VENDOW - Algerian E-commerce Ecosystem

VENDOW is a comprehensive digital ecosystem specifically designed for the Algerian market. It connects shop owners (Vendors), creative professionals (Freelancers), and local Shoppers into a single, seamless platform. Built with a focus on trust, ease of use, and local logistics, VENDOW empowers Algerian businesses to transition from physical storefronts to digital powerhouses.

## 🚀 Vision
**"Built by Algerians, for the Algerian market."**
We understand the unique challenges of local commerce. Our platform supports local payment methods, adapts to regional logistics, and celebrates the richness of Algerian store culture.

---

## 🏗️ Project Structure

The project is divided into two main components:
- **Front-end**: A collection of responsive, brand-aligned portals for different user roles.
- **Back-end**: A robust Node.js/Express API following the MVC architecture.

### Portals
- **🏠 Home Page**: The central hub showcasing our mission and entry points for all users.
- **🛍️ Shopper Portal**: A secure environment for browsing stores, managing carts, and tracking orders.
- **🏢 Business (Vendor) Portal**: A management dashboard for shop owners to handle inventory, orders, and analytics.
- **🎨 Freelancer Portal**: A dedicated space for creative professionals to offer Branding, Photography, and Video services directly to vendors.

---

## ✨ Core Features

### For Shoppers
- **Local Discovery**: Search for stores by City and Wilaya.
- **Trust First**: Interact with verified sellers through secure communication channels.
- **Seamless Browsing**: Easy navigation through categorized catalogues.

### For Vendors
- **Digital Boutique**: Transform any physical shop into a professional online storefront.
- **Inventory Control**: Real-time stock management and order tracking.
- **Freelance Access**: Directly hire top-tier local talent to improve brand visuals.

### For Freelancers
- **Service Marketplace**: Offer creative services directly to shop owners who need them.
- **Secure Payments**: Protected project payments and escrow-like handling.
- **Portfolio Showcase**: Display work specifically tailored to the e-commerce niche.

## 🛠️ Tech Stack

### Frontend
- **Structure**: Semantic HTML5
- **Styling**: CSS3 with **Bootstrap 5.3** and custom Brand Design System.
- **Interactivity**: Vanilla JavaScript (ES6+).
- **Aesthetics**: Premium Glassmorphic elements and brand-consistent gradients.

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js (v5+)
- **Database**: MongoDB with Mongoose ODM.
- **Authentication**: JSON Web Tokens (JWT) & Bcrypt for hashing.
- **Validation**: Express-validator for secure API inputs.

## 📦 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB (Local or Atlas instance)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-repo/team-6.git
   cd team-6
   ```

2. **Backend Setup**:
   ```bash
   cd back-end
   npm install
   # Create a .env file based on .env.example
   npm run dev
   ```

3. **Frontend Setup**:
   - The frontend consists of static files. You can serve them using any static file server or simply open the `Front-end/home/index.html` in your browser.
